# Cortex Domains Repository

## *Overview*

This repository is dedicated to defining and maintaining **Domain entities** in **Cortex**.
It also manages all **custom entities** used across the software catalog.
It acts as the authoritative source for how the organization models its **high-level business
domains** for discoverability, ownership, and architectural clarity.

Each domain is represented as a lightweight OpenAPI YAML file enriched with Cortex metadata.

More info here:
https://docs.cortex.io/configure/gitops#gitops-repository-structure
